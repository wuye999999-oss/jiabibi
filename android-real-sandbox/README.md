# 价比比真实账号沙盒 · 简洁版

目标：让用户在自己手机上的 WebView 里真实登录淘宝/京东/拼多多，然后本地读取商品标题、价格、店铺、规格、优惠和链接。

原则：

- 不保存账号密码。
- 不上传 cookie。
- 登录态只保存在用户本机 WebView。
- 第一阶段只做商品页读取，不做自动下单、不做绕风控。

## 打开方式

用 Android Studio 打开本目录：

```text
android-real-sandbox
```

然后运行 `app`。

## 简洁版界面

主界面只保留 6 个明显动作：

1. 输入框：输入关键词或粘贴商品链接。
2. 淘宝。
3. 京东。
4. 拼多多。
5. 读取价格。
6. 复制结果。

高级功能改成长按：

- 长按顶部标题：清除本机登录态。
- 长按“读取价格”：诊断页面。
- 长按“复制结果”：复制完整 JSON。
- 长按结果区域：清空读取结果。
- 手机返回键：WebView 返回上一页。

## 测试步骤

1. 输入 `小米充电宝`。
2. 点“京东”，进入京东搜索页，点进一个商品，点“读取价格”。
3. 输入同样关键词，点“淘宝”，登录后点进商品，点“读取价格”。
4. 点“拼多多”，进入商品页后点“读取价格”。
5. 点“复制结果”。

如果某个平台读不到价格，长按“读取价格”做诊断，再长按“复制结果”复制 JSON 发给 AI 继续优化。
