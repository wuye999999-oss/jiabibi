# 价比比真实账号沙盒 v4 · 第一性原理版

核心判断：用户真正要比较的不是平台 API 返回值，而是“自己账号、自己设备、自己页面上真实看到的价格”。

所以 v4 只围绕 3 个事实做：

1. 本机真实页面。
2. 页面真实价格。
3. 多平台同款谁更便宜。

原则：

- 不保存账号密码。
- 不上传 cookie。
- 登录态只保存在用户本机 WebView。
- 不造价格，不用假数据。
- 第一阶段只做商品页读取，不做自动下单、不做绕风控。

## 打开方式

用 Android Studio 打开本目录：

```text
android-real-sandbox
```

然后运行 `app`。

## v4 界面

主界面仍然只保留 6 个明显动作：

1. 输入框：输入关键词或粘贴商品链接。
2. 淘宝。
3. 京东。
4. 拼多多。
5. 读取价格。
6. 复制结果。

高级功能继续藏到长按：

- 长按顶部标题：清除本机登录态。
- 长按“读取价格”：诊断页面。
- 长按“复制结果”：复制完整 JSON。
- 长按结果区域：清空读取结果。
- 手机返回键：WebView 返回上一页。

## v4 新增

- 每个平台只保留最新一次读取结果，避免重复堆积。
- 自动从读取文本里提取数字价格。
- 自动比较最低价。
- 结果顶部直接显示“最便宜：平台 + 价格”。
- JSON 中增加 `best` 字段，后续可直接接入结果页。

## 测试步骤

1. 输入 `小米充电宝`。
2. 点“京东”，进入京东搜索页，点进一个商品，点“读取价格”。
3. 输入同样关键词，点“淘宝”，登录后点进商品，点“读取价格”。
4. 点“拼多多”，进入商品页后点“读取价格”。
5. 看顶部“最便宜”。
6. 点“复制结果”。

如果某个平台读不到价格，长按“读取价格”做诊断，再长按“复制结果”复制 JSON 发给 AI 继续优化。
