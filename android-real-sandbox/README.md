# 价比比真实账号沙盒 v3

目标：让用户在自己手机上的 WebView 里真实登录淘宝/京东/拼多多，然后本地读取商品标题、价格、店铺、规格、优惠和链接。

原则：

- 不保存账号密码。
- 不上传 cookie。
- 登录态只保存在用户本机 WebView。
- 第一阶段只做商品页读取，不做自动下单、不做绕风控。

## 打开方式

用 Android Studio 打开本目录：

```text
android-real-sandbox
```

然后运行 `app`。

## v3 功能

1. 淘宝 / 京东 / 拼多多快捷按钮。
2. 输入关键词时按平台打开对应搜索页。
3. 输入商品链接时直接打开商品页。
4. 用户自己在 WebView 里登录。
5. 到商品详情页后点击“读取价格”。
6. 平台专用选择器：淘宝、京东、拼多多分别增强标题/价格读取。
7. 本地 JS 读取：平台、标题、价格、券后/活动价、规格、店铺、图片、链接、UA。
8. 多次读取结果会汇总成多平台比价卡片。
9. 支持“诊断页面”：读取正文长度、价格节点数量、图片数量、正文样例。
10. 支持复制诊断信息，方便截图/发给 AI 修复。
11. 支持复制完整 JSON，便于后续接入结果页。
12. 支持清除读取结果，不影响登录态。
13. 支持清除本机 WebView Cookie / 缓存 / 历史。
14. 支持返回上一页。

## 测试步骤

1. 输入 `小米充电宝`。
2. 点“京东”，进入京东搜索页，点进一个商品，点“读取价格”。
3. 回到输入框输入同样关键词，点“淘宝”，登录后点进商品，点“读取价格”。
4. 点“拼多多”，进入商品页后点“读取价格”。
5. 如果某个平台读不到价格，点“诊断页面”。
6. 点“复制诊断”或“复制JSON”，把结果发给 AI 继续优化选择器。
