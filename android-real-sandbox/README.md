# 价比比真实账号沙盒 v5 · 买最低价

第一性原理：用户不是来研究接口、诊断、报表的。用户用了价比比，就是有明确需求：

1. 我想买这个东西。
2. 我想知道哪里最便宜。
3. 我想直接去最低价平台购买。

所以 v5 的主流程变成：

```text
选平台 → 读取价格 → 买最低价
```

原则：

- 不保存账号密码。
- 不上传 cookie。
- 登录态只保存在用户本机 WebView。
- 不造价格，不用假数据。
- 不自动下单，只把用户带回最低价商品页，由用户自己确认购买。

## 打开方式

用 Android Studio 打开本目录：

```text
android-real-sandbox
```

然后运行 `app`。

## v5 界面

主界面只保留 6 个明显动作：

1. 输入框：输入关键词或粘贴商品链接。
2. 淘宝。
3. 京东。
4. 拼多多。
5. 读取价格。
6. 买最低价。

高级功能继续藏到长按：

- 长按顶部标题：清除本机登录态。
- 长按“读取价格”：诊断页面。
- 长按“买最低价”：复制完整 JSON。
- 长按结果区域：清空读取结果。
- 手机返回键：WebView 返回上一页。

## v5 新增

- “复制结果”改成“买最低价”。
- 自动打开当前最低价商品链接。
- 用户确认后在平台内自己下单。
- JSON 中增加原则说明：真实页面、真实价格、直接购买路径。

## 测试步骤

1. 输入 `小米充电宝`。
2. 点“京东”，进入京东搜索页，点进一个商品，点“读取价格”。
3. 输入同样关键词，点“淘宝”，登录后点进商品，点“读取价格”。
4. 点“拼多多”，进入商品页后点“读取价格”。
5. 看顶部“最便宜”。
6. 点“买最低价”，回到最低价商品页。

如果某个平台读不到价格，长按“读取价格”做诊断，再长按“买最低价”复制 JSON 发给 AI 继续优化。
