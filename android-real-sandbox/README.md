# 价比比真实账号沙盒 v1

目标：让用户在自己手机上的 WebView 里真实登录淘宝/京东/拼多多，然后本地读取商品标题、价格、店铺和链接。

原则：

- 不保存账号密码。
- 不上传 cookie。
- 登录态只保存在用户本机 WebView。
- 第一版只做商品页读取，不做自动下单、不做绕风控。

## 打开方式

用 Android Studio 打开本目录：

```text
android-real-sandbox
```

然后运行 `app`。

## v1 功能

1. 输入商品链接或关键词。
2. 在 App 内 WebView 打开平台页面。
3. 用户自己登录。
4. 到商品详情页后点击“读取价格”。
5. 本地 JS 读取：平台、标题、价格、店铺、链接。
